# MCTS no textual do jogo da velha

Parte da palestra que eu dei na Twitch no dia 25 de maio de 2025.

## Como executar?

1. Faça um clone do repositório.
2. Acesse a pasta `mcts`.
3. Rode `poetry shell`.
4. Acesse a pasta `src`.
5. Rode `python -m mcts.main`.
